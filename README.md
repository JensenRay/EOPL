# EOPL Notes

This project is a personal record of studying *Essentials of Programming Languages* (EOPL).

The implementations here are incomplete and should be treated as learning exercises rather than finished language implementations. The goal is to explore ideas from the book step by step, keep experiments in one place, and evolve the code as new chapters and exercises are studied.

## Project Structure

- `src/`: Rust source for the current interpreter implementation.
- `tests/`: integration tests and sample EOPL programs used by the Rust project.
- `Cargo.toml`: Rust package manifest.
- `target/`: local Rust build artifacts.

The current codebase includes a small interpreter pipeline with tokenizing, parsing, and evaluation modules. It is expected to change as the study notes grow.
